### カット、コピーキー対応
範囲選択モード中に下記のキーを押すと、選択モードを終了してそれぞれのコマンドを実行するようにした。

| キー操作１ | キー操作２     | コマンド                          |
| ---------- | -------------- | --------------------------------- |
| ctrl + c   | ctrl  + insert | editor.action.clipboardCopyAction |
| ctrl + x   | shift + delete | editor.action.clipboardCutAction  |
